# Linux Basics Tasks

This repository contains solutions for basic Linux tasks, including file management, viewing contents, searching, compression, downloading, permissions, and environment variables.

## Structure

```
linux-basics-tasks/
├── README.md
├── linux_tasks.docx
└── Images/
    ├── Creating and Renaming Files-Directories.png
    ├── Viewing File Contents.png
    └── ...
```

## Tasks Covered

1. Creating and Renaming Files/Directories
2. Viewing File Contents
3. Searching for Patterns
4. Zipping and Unzipping
5. Downloading Files
6. Changing Permissions
7. Working with Environment Variables

See `linux_tasks.docx` for detailed documentation and screenshots.
